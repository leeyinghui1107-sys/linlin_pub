# 林霖智能监控系统

林霖智能监控系统是一套面向数据中心、机房、弱电间、通信站点、边缘站点和中小型园区的动力环境监控与动环监控系统。系统以单机部署为主，在一个统一的 Web 平台中完成设备接入、数据采集、实时展示、历史保存、告警处理、通知联动、运行监控和基础配置管理。

它适合需要本地部署、低依赖、快速交付和长期稳定运行的现场监控项目。

## 主要用途

- 数据中心、机房、弱电间的动力环境监控
- 通信站点、边缘节点、园区设备的集中监测
- UPS、空调、电源、传感器、网络设备等现场设备接入
- 实时数据查看、历史追溯、告警确认和事件回溯
- 短信、声光等告警通知与联动
- 作为标准监控产品交付，或通过插件扩展对接第三方平台

## 核心能力

- **设备接入**：支持串口、TCP Client、TCP Server、SNMP 主动采集，适配 Modbus、SNMP、PMBus 等常见协议。
- **实时监控**：通过 Web 页面查看设备状态、实时值、历史值、小时汇总和告警事件。
- **告警闭环**：支持属性告警、设备中断告警、端口中断告警，以及确认、解除、历史查询和事件回溯。
- **数据存储**：基于 Go + SQLite，配置数据与运行数据分离，支持历史保存、备份、清理和月度维护。
- **运维管理**：提供告警中心、事件日志、运行指标、配置管理、端口管理、设备列表和设备详情页面。
- **插件扩展**：可扩展 MQTT 上传、SNMP Agent、第三方平台接口、定制化通知和行业集成能力。
